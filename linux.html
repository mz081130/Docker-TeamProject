Basic Linux Commands

  1. pwd- To know the directory/absolute path of where you are currently
  2. ls- To see files in current directory, to see list of all the files use ls -a
  3. cd - To go into a directory. e.g., cd documents and cd.. to go back a folder
  4. mkdir & rmdir- Use mkdir "foldername" to creat a new filer and rm dir "foldername" to delete a folder
  5. touch "filename.extension"- To create a new file in your current directory
  6. cp- To copy files from current directory to the desired directory. For example, cp "filename.ext" /home/documents/tutorial
  7. echo- It helps to move some data, usually text into a file. For example,if you want to create a new text file or add to an already made text file, you just need to type in,echo testing, 123 >> newfile.txt
  8. sudo- stands for "SuperUser Do". So, if you want any command to be done with administrative or root privileges
  9. cat- Use the cat command to display the contents of a file
  10. apt-get- Use apt to work with packages in the Linux command line. Use apt-get to install packages
  11. clear-To clear the terminal if it gets filled up with too many commands

Users can use the following to log into an already created docker container

$docker exec -it "container name" /bin/bash
